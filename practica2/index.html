Arduino Electrónica

- Resumen -

En esta práctica continuamos realizando diferentes conexiones con Arduino que iban desde lo más simple a lo más complejo, el propósito fue finalizar el curso de arduino básico II desde la plataforma de Arduino con ayuda de la simulación en Tinkercad proporcionado por el profesor.

- Introducción -

El Arduino es una plataforma que consiste en una placa electrónica con un microcontrolador programable y una serie de pines de conexión, dicha herramienta nos permite a escribir y cargar códigos en la placa para automatizar y controlar diferentes dispositivos. Se utiliza principalmente en la electrónica, robótica, domótica, etc.

La programación con Arduino consiste en escribir y carga códigos a la placa, se usa una plataforma de código abierto que utiliza un lenguaje de programación basado en C/C++. Permite crear proyectos simples o complejos, utilizando sensores y actuadores. Gracias a la programación con Arduino es posible leer datos en sensores, controlar dispositivos físicos en tiempo real, realizar cálculos, etc.

- Conclusiones -
